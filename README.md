In-memory Wander Join Implementation For some TPC-H Queries
====================================

## Build
For release version:

    make release

For debug version:

    make debug

## Run

TPC-H data are assumed to be placed under data/. The
path of the data files relative to CWD is assumed to be ../data/*.tbl.
